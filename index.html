Варианты работы с приложением

Рассмотрим, кто и как может использовать приложение

  • Принципы работы с ООО:

1. Регистрация ККТ (в нашем случае Ferma)
2. Регистрация в ATI.su
3. Получение приложения "Три кота"
4. Добавление формы расчёта с помощью ККТ Ferma (пункт1)
5. При завершении сделки формируется чек и направляется в ОФД

  • Принцип работы с ИП:

1. Регистрация ККТ (в нашем случае Ferma)
2. Регистрация в ATI.su
3. Получение приложения "Три кота"
4. Добавление формы расчёта с помощью ККТ Ferma (пункт1)
5. При завершении сделки формируется чек и направляется в ОФД

  • Принцип работы с самозанятым:

1. Регистрация в ATI.su
2. Получение приложения "Три кота"
3. Добавление формы расчёта с помощью QR кода приложении банка или наличными деньгами
4. При завершении сделки формируется QR код для перевода на ваши реквизиты

  • Принцип работы с физическим лицом:

1. Регистрация в ATI.su
2. Получение приложения "Три кота"
3. Добавление формы расчёта с помощью QR кода приложении банка или наличными деньгами
4. При завершении сделки формируется QR код для перевода на ваши реквизиты

ПОЧЕМУ МЫ?

Выделим пару возможностей
нашего приложения

Облачная касса (ОФД Ферма)

Никаких физически касс, нужно только наше ПО и подписка ОФД Ферма

Разные виды оплат

Получайте деньги за ваши грузы наличными/ картой (прикладывайте карту к телефону NFC), сканируйте QR Code

Окно статистики

Наблюдайте на свои результаты через наше ПО

Соотвествие закону

Работайте согласно закону, наше ПО будет создавать накладные согласно ЭтРН
